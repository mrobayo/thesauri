
## Bower
[Getting started](http://blog.teamtreehouse.com/getting-started-bower)
 
### Instalar paquete
  bower install components-font-awesome --save 
### Quitar paquete
  bower uninstall components-font-awesome
  
  
  
## Composer
   Comandos de composer
